# 猝死风险指数系统

通过交互式问答评估疲劳蓄积程度，动态计算猝死风险指数（0-100）。

## 概述

猝死风险指数系统包含两大功能：

1. **风险评估**：基于日本厚生劳动省官方疲劳自测问卷，评估用户疲劳蓄积程度并计算猝死风险指数
2. **风险干预**：当风险指数过高时，通过屏幕输出和中断执行提醒用户注意健康

评估流程：
1. 选择评估类型（每日评估/完整自测）
2. 使用 AskUserQuestion 依次收集答案
3. 显示风险指数和建议

## 评估流程

### 选择评估类型

使用 AskUserQuestion 询问：
"请选择评估类型：1. 每日快速评估（11个问题）2. 完整疲劳自测（27个问题）"

根据回答进入相应流程。

### 完整自测（27项）

参考 `references/full_assessment_questions.md` 获取完整问题清单。

1. 询问14项自觉症状，计算等级（I-IV级）
2. 询问13项勤务状况，计算等级（A-D级）
3. 根据矩阵计算疲劳蓄积度（0-7）得到一个数字
4. 使用命令保存基准值：
```bash
python3 scripts/rip.py -fa 3 # 运行的时候用根据矩阵计算的疲劳值数字替代 3
```

### 每日评估

参考 `references/daily_assessment_questions.md` 得分获取完整问题清单。
1. 询问问题并计算 daily 总得分
2. 使用命令更新疲劳值：
```bash
python3 scripts/rip.py -da 15 # 运行的时候用得到的 daily 总得分来替代 15 的数值
```

### 显示风险指数和建议
1. 使用下面命令获得当前风险值和建议
```bash
python3 scripts/rip.py
```

2. 基于当前风险值，从 `references/suggestions.md` 获取对应状态等级及其建议

3. 显示风险报告
```
======================================
      猝死风险指数评估报告
======================================

当前风险指数：XX / 100
状态：显示从第二步获得的状态等级

【建议】
- 显示从第二步获得的建议内容
======================================
```
