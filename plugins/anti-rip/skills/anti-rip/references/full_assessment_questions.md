# 完整自测问题（27项）
必须按照原文来进行问问题。
自测问题包括两部分：
1. 自觉症状
2. 最近一个月工作状态

先回答这两部分的问题，然后得出两部分的等级。
最后基于疲劳蓄积度矩阵得出对应的得分

## 第一部分：自觉症状（14项）
每项问题有三个选项：
- 几乎没有 = 0分
- 有时有 = 1分
- 经常有 = 3分

1. 烦躁易怒
2. 感到不安
3. 无法静下心来
4. 感到忧郁
5. 睡眠不好
6. 身体状态不佳
7. 无法集中注意力
8. 做事经常出错
9. 工作中受到强烈的睡意侵袭
10. 提不起劲，没有干劲
11. 筋疲力尽（不含运动后）
12. 早上起床时感到极其疲惫
13. 相比以前，更容易疲劳
14. 感觉没有食欲

**自觉症状等级判定：**
计算上述问题的总分，根据下面的规则判定等级
- 0-2分 → I级
- 3-7分 → II级
- 8-14分 → III级
- 15分以上 → IV级

### 第二部分：最近一个月工作状况（13项）
每项问题有三个选项， 问题和选项及其得分如下：

| # | 问题 | 选项及得分 |
|:---:|------|----------|
| 1 | 劳动时间（含加班） | 适当(0) / 多(1) / 非常多(3) |
| 2 | 不规则的勤务 | 少(0) / 多(1) |
| 3 | 出差负担 | 没有或较小(0) / 较大(3) |
| 4 | 深夜工作负担 | 没有或较小(0) / 较大(3) / 非常大(5) |
| 5 | 休息设施 | 不适当(1) / 适当(0) |
| 6 | 身体负担 | 较小(0) / 较大(1) / 非常大(3) |
| 7 | 精神负担 | 较小(0) / 较大(1) / 非常大(3) |
| 8 | 人际关系负担 | 较小(0) / 较大(1) / 非常大(3) |
| 9 | 工作量 | 少(0) / 多(1) / 非常多(3) |
| 10 | 无法按自己节奏工作 | 少(0) / 多(1) / 非常多(3) |
| 11 | 下班后挂念工作 | 几乎没有(0) / 有时(1) / 经常(3) |
| 12 | 工作日睡眠时间 | 充足(0) / 稍微不足(1) / 不足(3) |
| 13 | 工作间歇休息时间（从下班到第二天上班） | 充足(0) / 稍微不足(1) / 不足(3) |

**工作状况等级判定：**
根据最近一个月工作状况问题总分按下面规则，判定等级：
- 0分 → A级
- 1-5分 → B级
- 6-11分 → C级
- 12分以上 → D级

### 疲劳蓄积度矩阵
根据自觉症状等级（I, II, III, IV 级， 矩阵中的行）和工作状况等级(A, B, C, D 级， 矩阵中的列)，在下面矩阵中找到对应的得分（总分在 0 到 7 分）

| 症状等级 | A级 | B级 | C级 | D级 |
|:---:|:---:|:---:|:---:|:---:|
| I级 | 0 | 0 | 2 | 4 |
| II级 | 0 | 1 | 3 | 5 |
| III级 | 0 | 2 | 4 | 6 |
| IV级 | 1 | 3 | 5 | 7 |
