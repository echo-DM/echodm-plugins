# 每日快速评估问题清单（9项）

## 问题及得分
问题有如下等级 p0, p1, p2, p3.

使用 AskUserQuestion 依次询问 p0, p1, p2, p3 问题, 计算总得分。

### p0 急性硬件故障（一票否决，直接熔断）
问问题的时候严格保持问题和选项的文本如下

| # | 问题 | 选项及得分 |
|:---:|------|----------|
| 1 |胸闷与痛觉预警 | A. 完全没有胸部不适(+0) / B. 偶尔觉得胸口有些发闷，深呼吸或休息几分钟后能完全缓解(+10) / C. 突发胸口严重的压迫感（像压了巨石），或出现不明原因的一身冷汗、剧烈胃痛、甚至放射到左肩/下巴(+100) |
| 2 | 心跳与供血预警 | A. 感受不到自己心脏的跳动(+0) / B. 偶尔起步或站起过猛时，眼前有一点发黑(+3) / C. 安静坐着时，突然觉得心脏像拖拉机一样狂跳/漏跳一拍，或者近期频繁出现眼前全黑、甚至差点晕倒(+100) |

| # | 问题 | 选项及得分 |
|:---:|------|----------|

| 2 | 躯体化抗拒 | A. 正常面对工作，顶多心理上偶尔吐槽(+0) / B. 休息日总觉得手机在震动、钉钉在响（幻震），容易烦躁(+5) / C. 产生极其严重的躯体化抗拒：一想到要开电脑、发版或看到工作消息，就控制不住地心慌、胃部抽紧、呼吸变浅(+10) |
| 3 | 咖啡因耐受 | A. 不依赖咖啡/烟草，或者每天固定喝一点，提神效果正常(+0) / B. 依赖度明显增加，需要靠加倍喝浓咖啡/抽烟来维持清醒(+5) / C. 彻底耐受：狂喝红牛/咖啡也提不了神，大脑发木，但身体却会出现心慌、手抖、胃酸反流(+10) |
| 4 | 久坐 | A. 单次久坐不超过 1 小时(+0) / B. 单次久坐超过 1 小时，不超过 2 小时(+5) / C. 单次久坐超过 2 小时(+10) |
