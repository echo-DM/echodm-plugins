# anti-rip 猝死风险评估插件

## 引言
 我们 Vibe Coding 的时候，AI 就像大脑的延伸，在后台狂奔。那一刻，多巴胺疯狂分泌，你感觉自己像是一个掌控赛博宇宙的上帝。
**但问题是：上帝是不需要睡觉的，而你需要。**
一个残酷的悖论是生产力工具越强，离 AGI 越近，我们离猝死反而越近。

程序员是创造系统、维护系统的工程师，我们的身体何尝不是一个系统。任何系统运行久了都会有摩擦、有 Bug。头痛脑热在所难免，关键是绝不能搞出 P0 级别的 Bug——把人直接给"物理宕机"了。

**猝死的本质** —— 80% 都是心脏问题，而心脏问题的 80% 都是冠状动脉问题（冠心病）。可以把冠状动脉想象成一个"皮薄馅大"的饺子：随着熬夜、压力，包裹斑块的纤维帽（饺子皮）会越来越薄。一旦皮破了——熬夜、久坐、情绪激动，随时可能"吧嗒"一声。

所以，光知道原理不够，你得有 On-Call 的报警机制。

anti-rip 反猝死插件是给你的身体加一个 Monitor。

---

## 功能介绍
### 功能一：风险评估（Know Your RIP Index）

基于 **日本厚生劳动省** 官方疲劳自测问卷——这是全球唯一一个由国家级机构制定的过劳程度自测标准，来自过劳死历史悠久的国家。

通过初始的交互问答，它会确定你基本的疲劳状况；紧接着是动态微调，监控你的日常行为（昨晚 Commit 到了几点？这周 on call 了几次？今天久坐是否超过 4 小时？）——每一个高危行为，都会让你的 RIP index 指数上升。

### 功能二：风险干预（When Risk Gets High）

当你沉浸在 Vibe Coding 的快感中，或者处于被 Deadline 追杀的恐慌中时，你的大脑会自动屏蔽掉身体发出的 Error 警报。Claude Code 会实时显示你的 RIP 进度，不断提醒你身体情况。当风险指数过高时，通过屏幕输出警报。

---

## 风险等级

| 风险值 | 等级 | 说明 |
|--------|------|------|
| 0-40 | 低风险 | 身体状态良好，继续保持 |
| 41-60 | 中风险 | 身体已出现疲劳信号，需要调整 |
| 61-80 | 高风险 | 身体透支，注意休息 |
| 81-99 | 高危风险 | 身体严重透支，需立即干预 |
| 100 | 临界区 | 生死时刻，立即就医 |

> ⚠️ **P0 级警报**：如果近期熬夜后感到胸口像被大石头压住、左手臂发麻、牙痛——**这不是疲劳，这是你的硬件在报 P0 级故障了！**

## 安装方法

### 方式一：Claude Code 插件市场安装（推荐）

```bash
/plugin marketplace add echo-DM/echodm-plugins
/plugin install anti-rip@echodm-plugins
```

### 方式二：本地加载（开发、测试用）

```bash
# 直接加载本地插件
claude --plugin-dir /path/to/echodm-plugins/plugins/anti-rip
```

> **提示**：查看插件是否加载成功，运行 `/anti-rip` 或说"猝死风险评估"，能正常响应即表示安装成功。

## 使用方法

### 在 Claude Code 中使用

直接告诉 Claude Code：
- "猝死风险评估"
- "疲劳自测"
- "健康风险检查"
- "评估我的健康风险"
- 或使用命令 `/anti-rip`

系统会引导你完成：
1. 选择评估类型（每日评估/完整自测/重置）
2. 回答问卷问题
3. 查看风险指数和建议

### 对应脚本的作用

```bash
# 查看当前风险状态
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py

# 设置基准疲劳值（0-7），完整评估后使用
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py -fa 3

# 添加每日风险点（同时标记每日评估完成）
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py -da 15

# 检查评估完成状态
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py -e

# 重置所有疲劳数据
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py -r
```

### 自动提醒

每次启动 Claude Code 时，SessionStart hook 会自动显示你的当前风险指数。

## 数据来源

基准问卷采用日本厚生劳动省官方疲劳自测标准，包含：
- **14 项自觉症状**（I-IV 级）
- **13 项勤务状况**（A-D 级）
- [→ 完整自测问卷（27项）](./skills/anti-rip/references/full_assessment_questions.md)
通过矩阵计算得出疲劳蓄积度。

行为微调问卷从 P0 到 P3，监控危险行为或症状：
- **P0 急性硬件故障**（一票否决，直接熔断）：胸闷/胸痛/压迫感/心悸/心跳不规律/眼前发黑/晕厥/剧烈头痛 → 立即报警
- **P1 核心资源透支**：睡眠剥夺、深度疲劳感
- **P2 进程调度异常**：交感神经锁死、躯体化抗拒、咖啡因耐受、久坐
- **P3 内存泄漏与边缘报错**：短期记忆与专注力下降、局部免疫与消化异常、情绪控制力下降
- [→ 每日快速评估问卷](./skills/anti-rip/references/daily_assessment_questions.md)

[→ 分级建议与干预方案](./skills/anti-rip/references/suggestions.md)

## 注意事项

- 风险评估仅供参考，身体不适请及时就医
- 评估结果仅代表当前疲劳状态，不构成医疗诊断
- 如出现胸闷、心悸、眼前发黑等症状，请立即就医
