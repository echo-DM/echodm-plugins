# anti-rip 猝死风险评估插件

通过疲劳自测评估猝死风险指数，守护程序员健康。

## 功能介绍

### 风险评估
基于 **日本厚生劳动省** 官方疲劳自测问卷（全球唯一国家级过劳程度自测标准），评估您的疲劳蓄积程度并计算猝死风险指数（0-100）。

### 风险干预
当风险指数过高时，通过屏幕输出和中断执行提醒您注意健康。

## 风险等级

| 风险值 | 等级 | 说明 |
|--------|------|------|
| 0-40 | 低风险 | 身体状态良好，继续保持 |
| 41-60 | 中风险 | 身体已出现疲劳信号，需要调整 |
| 61-80 | 高风险 | 身体透支，注意休息 |
| 81-99 | 高危风险 | 身体严重透支，需立即干预 |
| 100 | 临界区 | 生死时刻，立即就医 |

## 使用方法

### 在 Claude Code 中使用

直接告诉 Claude Code：
- "猝死风险评估"
- "疲劳自测"
- "健康风险检查"
- 或使用命令 `/anti-rip`

系统会引导您完成：
1. 选择评估类型（每日评估/完整自测）
2. 回答问卷问题
3. 查看风险指数和建议

### 命令行使用

```bash
# 查看当前风险状态
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py

# 设置基准疲劳值（0-7），完整评估后使用
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py -fa 3

# 添加每日风险点
python3 plugins/anti-rip/skills/anti-rip/scripts/rip.py -da 15
```

## 数据来源

问卷采用日本厚生劳动省官方疲劳自测标准，包含：
- **14 项自觉症状**（I-IV 级）
- **13 项勤务状况**（A-D 级）

通过矩阵计算得出疲劳蓄积度。

## 注意事项

- 风险评估仅供参考，身体不适请及时就医
- 评估结果仅代表当前疲劳状态，不构成医疗诊断
- 如出现胸闷、心悸、眼前发黑等症状，请立即就医
